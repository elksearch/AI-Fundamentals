---

## Milestone by Milestone

### 1950 — Turing Test ("AI")
**Alan Turing** proposed the Turing Test:
- Can a machine convince a human it is also human?
- First formal definition of machine intelligence
- Set the philosophical foundation for all AI

```
If a human cannot distinguish
machine response from human response
→ Machine is considered intelligent
```

---

### 1956 — IBM 704 / LISP
- **IBM 704** — first computer used for AI research
- **LISP** programming language created for AI
- First AI programs written — rule based logic
- Term "Artificial Intelligence" officially coined at Dartmouth Conference

---

### 1960s — ELIZA & NLP Begins
- **ELIZA (1964-1966)** — world's first chatbot
  - Created at MIT by Joseph Weizenbaum
  - Pure pattern matching — no real understanding
  - Simulated a psychotherapist
  - People got emotionally attached to it — showed humans anthropomorphise machines easily
- **NLP begins** — first attempts at natural language processing
- Machines starting to handle human language (very primitively)

---

### 1970s — Prolog
- **Prolog** — logic programming language for AI
- Used for rule based expert systems
- Enabled more complex reasoning than LISP
- Still rule based — humans writing all the logic

---

### 1980s — Expert Systems
- Peak of rules-based AI
- **Expert Systems** — encoded human expert knowledge as rules
- Used in medicine, finance, engineering
- Example: MYCIN — diagnosed bacterial infections

```
IF patient has fever
AND patient has high white blood cell count
AND symptom is X
THEN diagnosis is Y
THEN recommend antibiotic Z
```

- **Limitation** — brittle, expensive to maintain, couldn't handle new situations
- Led to first "AI Winter" — funding dried up, hype died

---

### 1990s — Deep Blue (1997)
- **IBM Deep Blue** beats world chess champion Garry Kasparov
- Huge milestone — machine beats human at complex strategy game
- Still rule based + search algorithms — not ML
- Showed machines could exceed human performance in specific domains
- Public perception of AI changed forever

---

### 1997-2000s — Expert Systems Decline, ML Rises
- Expert systems proved too rigid for real world
- Machine Learning started gaining traction
- Statistical approaches replacing hand-coded rules
- Data becoming more available — internet growing
- Algorithms: SVM, Decision Trees, Random Forest emerge

---

### 2011 — Watson wins Jeopardy!
- **IBM Watson** defeats human champions at Jeopardy!
- Jeopardy requires understanding language, context, wordplay, trivia
- Far harder than chess — needs NLP + reasoning
- Watson used ML + NLP + massive knowledge base
- Marked mainstream acceptance of AI in complex language tasks

---

### 2010s — ML/DL Revolution
- **Deep Learning** takes off
- **2012** — AlexNet wins ImageNet competition by massive margin
  - Neural networks suddenly outperform everything else at image recognition
- **2014** — GANs (Generative Adversarial Networks) invented
- **2016** — AlphaGo beats world Go champion
  - Go has more positions than atoms in universe
  - Pure rules-based impossible — needed DL + Reinforcement Learning
- **2017** — Transformer architecture introduced
  - Google paper: "Attention Is All You Need"
  - Changed everything

---

### 2020s — GenAI & Agents

#### 2020
- **GPT-3** released — 175B parameters
- First time AI writing was genuinely impressive
- Few-shot learning — AI learns from just a few examples

#### 2022
- **ChatGPT** launched — changes public perception forever
- 100 million users in 2 months — fastest product adoption in history
- **Stable Diffusion, DALL-E 2** — image generation goes mainstream

#### 2023
- **GPT-4** — multimodal, reasoning improvements
- **Llama** — Meta releases open source LLM
- **Claude** — Anthropic launches safety-focused LLM
- **Gemini** — Google's response

#### 2024
- **Agentic AI** takes centre stage
- **MCP** — Model Context Protocol standardised
- **Reasoning Models** — o1, DeepSeek R1
- **Multimodal** becomes standard

#### 2025
- **DeepSeek** shocks industry — frontier model at 1/30th cost
- **AI Agents** becoming production ready
- **GenAI** embedded in every enterprise workflow

## The Exponential Curve — Why Now?

Three things converged in 2010s to cause explosion:

```
1. Data        → Internet generated massive training data
                  Billions of labelled images, text, code

2. Compute     → GPU costs dropped dramatically
                  Cloud computing made GPUs accessible

3. Algorithms  → Transformers, attention mechanism
                  Made training large models possible
```

All three were needed simultaneously. Before 2010 — missing pieces.
